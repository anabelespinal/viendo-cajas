Laboratoria

Laboratoria ensena a jovenes mujeres a crear productos digitales con el proposito de insertarlas en la creciente economia del Peru. en un periodo de 6 meses nuestras coders aprenden:

  1. El lenguaje del futuro

    La programacion esta detras de todos los productos digitales que usamos dia a dia. Nuestras coders aprenden HTML, CSS y Javascript, terminando preparadas para crear infinitos productos y soluciones web.

  2. Aprender a aprender

    Para triunfar como desarrolladoras web nuestras coders se convierten en autodidactas, descubriendo que no hay limites para seguir creciendo en este campo.
