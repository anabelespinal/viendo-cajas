Laboratoria

Laboratoria enseña a jóvenes mujeres a crear productos digitales con el propósito de insertarlas en la creciente economía del Perú. en un periódo de 6 meses nuestras coders aprenden:

  1. El lenguaje del futuro

    La programación esta detrás de todos los productos digitales que usamos día a día. Nuestras coders aprenden HTML, CSS y Javascript, terminando preparadas para crear infinitos productos y soluciones web.

  2. Aprender a aprender

    Para triunfar como desarrolladoras web nuestras coders se convierten en autodidactas, descubriendo que no hay límites para seguir creciendo en este campo.
